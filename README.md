# Multimedia Video Transcription System

## Описание

Данный программный продукт представляет собой веб-сервис для автоматизированного преобразования аудиодорожек видео в текстовую форму. Система позволяет пользователям загружать видеофайлы через веб-интерфейс, после чего извлекается аудио и выполняется автоматическое распознавание речи с последующей выдачей текстовой транскрипции.

Фронтенд написан на **React**, обеспечивая удобный и интуитивно понятный интерфейс для загрузки видео и отображения результатов. Бэкенд реализован на **FastAPI** .
## Функциональные возможности

Система позволяет:
- Загружать видеофайлы через веб-интерфейс.
- Автоматически извлекать аудиодорожку из видео.
- Выполнять распознавание речи и преобразование аудио в текст с использованием нейронной сети.
- Отображать транскрипцию в веб-интерфейсе.
- Вести логирование обработки и ошибок для диагностики.

## Минимальные системные требования

- **ОС:** macOS, Windows или Linux.  
- **Процессор:** 4 ядра или более.  
- **Оперативная память:** 8 ГБ и выше.  
- **Дисковое пространство:** минимум 10 ГБ.  
- **Интернет:** требуется для взаимодействия с веб-интерфейсом и загрузки видео.  

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone <URL_репозитория>
cd <название_проекта>
```


## Установка и запуск бэкенда (FastAPI + PyTorch)
Перейдите в директорию сервера:
```bash
    cd nn
```
Создайте виртуальное окружение и активируйте его:
```bash
python -m venv venv
source venv/bin/activate   # для macOS/Linux
venv\Scripts\activate      # для Windows
```
Установите зависимости:
```bash
pip install -r requirements.txt
```
Запустите сервер:
```bash
uvicorn main:app --reload
```
Бэкенд будет доступен по адресу http://127.0.0.1:8000.

## 3. Установка и запуск фронтенда (React)
Перейдите в директорию фронтенда:
```bash 
cd front
```
Установите зависимости:
```bash
npm install
```
Запустите фронтенд:
```bash
npm start
```
Приложение откроется в браузере по адресу http://localhost:3000.
# Использование
- Откройте веб-приложение.
- Загрузите видеофайл (поддерживаемые форматы: MP4, MOV).
- Дождитесь завершения обработки.
- Получите текстовую транскрипцию аудиодорожки.
## Ограничения и рекомендации
Качество распознавания зависит от разборчивости речи и уровня фонового шума.
Размер видеофайла не должен превышать ограничения сервера (рекомендуется до 500 МБ).
Для корректной работы сервер должен иметь доступ к GPU (рекомендуется для больших файлов и ускорения распознавания).
